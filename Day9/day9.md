# Day 8 – NutriScope: MVP to Enhanced Application

## Prompt 1 – Build MVP

### Objective
Create a complete single-file HTML nutrition tracking application called **NutriScope**.

### Features Generated
- User Profile Inputs
  - Age
  - Gender
  - Height
  - Weight
  - Activity Level
  - Dietary Preference

- Food Logging
  - Add Food
  - Quantity & Unit
  - Editable Table
  - Remove Entry

- Food Database
  - 20 Common Foods

- Nutrition Tracking
  - Calories
  - Protein
  - Carbohydrates
  - Fat
  - Fiber
  - Iron
  - Calcium
  - Vitamin C
  - Vitamin D
  - Vitamin B12

- Dashboard
  - Energy Progress
  - Macro Chart
  - Nutrient Summary
  - Top Deficiencies
  - Top Excesses

- Recommendations
  - Food Additions
  - Food Swaps
  - Portion Adjustments

- UI
  - Premium SaaS Design
  - Dark Theme
  - Mobile Responsive
  - Chart.js
  - Single HTML File
  - No Backend

---

# Prompt 2 – Enhancement

The second prompt upgraded the existing NutriScope application instead of rebuilding it.

### New Features Added
- CSV Upload Support
- 40 Additional Food Items
- Expanded Micronutrient Tracking
- 2-Day Meal Planner
- Nutrition Risk Analysis
- Educational Disclaimer
- Nutrition Information Sources
- Better & More Interactive Charts
- Advanced Personalized Recommendations

---

# Improvements from Prompt 1 → Prompt 2

| MVP (Prompt 1) | Enhanced Version (Prompt 2) |
|---------------|-----------------------------|
| 20 Foods | 60 Total Foods |
| Basic Micronutrients | Expanded Micronutrients |
| Food Logging | Food Logging + CSV Import |
| Basic Dashboard | Advanced Dashboard & Charts |
| Simple Recommendations | Personalized Smart Recommendations |
| Nutrition Tracking | Risk Analysis |
| Single Day Tracking | 2-Day Meal Planner |
| Basic Information | Nutrition Sources & Disclaimer |

---
## Key Learning

This exercise demonstrated the power of iterative prompting. Instead of creating a new application from scratch, the second prompt enhanced the existing project by adding advanced functionality, improving usability, and making the application more practical for real-world nutrition tracking.
