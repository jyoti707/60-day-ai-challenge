# Day 2 – Prompt Engineering

## Objective
Today I learned how a simple (lazy) prompt differs from a well-structured engineered prompt. I generated responses using both prompts in Claude AI and compared the results.

## What I Worked On
- Generated an image using a basic prompt.
- Generated another image using an engineered prompt.
- Compared both outputs.
- Observed improvements in clarity, structure, examples, and visual quality.

## Key Learnings
- Well-structured prompts produce more accurate and detailed responses.
- Adding context, objectives, formatting, and audience improves AI output.
- Prompt engineering helps create higher-quality images and explanations.
